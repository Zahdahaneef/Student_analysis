## 📌 Internship Details

* **Organization:** Maincraft Technology
* **Role:** Data Analysis Intern
* **Task:** Task 1 – Student Performance Analysis

---

## 📖 Project Overview

This project focuses on analyzing student performance data using Python.
The objective is to explore factors affecting student grades and visualize patterns in the dataset.

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

---

## 📂 Dataset

The dataset contains student-related information such as:

* Demographics (age, gender, address)
* Study habits (study time, failures)
* Family background (parent education)
* Academic performance (G1, G2, G3)

---

## 📊 Analysis Performed

* Data cleaning and preprocessing
* Statistical analysis
* Correlation analysis
* Group-based analysis (e.g., gender performance)

---

## 📈 Visualizations

* Histogram of grades (G1, G2, G3)
* Scatter plot (Study Time vs Final Grade)
* Bar charts (Gender comparison)

---

## 🔍 Key Insights

* Most students scored in the average range (10–15)
* Study time shows a positive relationship with performance
* Male students have slightly higher average scores than female student

## ✅ Conclusion

This project demonstrates basic data analysis and visualization using Python.
It helps in understanding how different factors influence student academic performance.
